Detailed Description

template<typename Index_, typename Data_, typename Distance_, class DistanceMetric_ = DistanceMetric<Data_, Distance_>>
class knncolle_kmknn::KmknnSearcher< Index_, Data_, Distance_, DistanceMetric_ >

KMKNN searcher.

Instances of this class are usually constructed using KmknnPrebuilt::initialize().

Template Parameters
Index_Integer type for the observation indices.
Data_Numeric type for the input and query data.
Distance_Floating-point type for the distances.
DistanceMetric_Class implementing the distance metric calculation. This should satisfy the knncolle::DistanceMetric interface.
