◆ Common

template<typename Data_ , typename Distance_ >
using knncolle_kmknn::Common = decltype(std::declval<Data_>() - std::declval<Distance_>())

Common type for promoting Data_ and Distance_ during subtraction. If Data_ is integer, the common type will be the floating-point type of Distance_; otherwise if both types are floating-point, the common type will be the higher-precision type. This is used to define the common datatype for storing both the input data and the centroids, so that the same DistanceMetric can be used to compute the distances to a query datapoint.

Template Parameters
Data_Numeric type for the input and query data.
Distance_Floating-point type for the distances.
